Project documentation tools

Usage:
dt [TOOL] <ARGS>

Tools:
	help - this help screen (alias: -h, --help)
	adr - create and manage ADRs (Architecture Decision Records)
	rdme - initialize and manage project top-level README.md (alias: readme)
	lcs - initialize and manage project LICENSE.txt (alias: license)
	chglg - initialize and manage project CHANGELOG.md (alias: changelog)

For specific tool help, try dt tool help. For an example:
	dt adr help
